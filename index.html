Pneumonia Detection Using Deep Learning

GitHub Repository: LokeshDragneel/Pneumonia-Detection-DL-Rpa

Project Overview

This project utilizes deep learning techniques, specifically the VGG16 and RetinaNet architectures, to classify chest X-ray images as either Pneumonia or Normal. It aims to assist medical professionals by automating the detection of pneumonia using convolutional neural networks.

Key Features

  • Uses pretrained VGG16 and RetinaNet models
  • High accuracy classification of chest X-ray images
  • Interactive notebooks for training and evaluation
  • Preprocessing and augmentation of image data
  • Saved model in .h5 format for deployment

How to Run

  1. Clone the repo: git clone https://github.com/LokeshDragneel/Pneumonia-Detection-DL-Rpa.git
  2. Install requirements: pip install -r requirements.txt
  3. Run the Jupyter Notebook or training script
  4. Evaluate model performance
View GitHub Repo

Model Results

The VGG16 model achieved approximately 96% accuracy on the test dataset, showing strong potential in real-world applications. Visualizations such as confusion matrices and training graphs are available in the notebooks.
